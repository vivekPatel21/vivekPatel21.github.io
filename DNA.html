Nth Length DNA sequencer

Tags: C++, recursion, and algorithms

This is a DNA sequencer that I programmed in C++ capable of reading any DNA sequence and returing a complimenting strand and spliting the DNA before printing the lists back out. It can sequence a DNA sequence of any length and will even check for minor problems such as a nucleotide not being correct, or there being a space in the strand, returning error messages as needed. For testing preferably you are going to need to provide a "DNA STRAND HERE" "4" for the DNA strand and separater. here or you can download it yourself here.

Guide

The program checks for 2 inputs, it checks for a DNA strand and a number to separate the strand and compliment into lists. Afterwards the program will check for a few things such as spaces and things that should not be in a DNA strand and than it will print out the strands in lists, the size of the lists is determined by the separater.

Technical Details

The time and space complexity of this program is O(n) where n is the size of the DNA Sequence. There are efforts to try to make the code more efficient an quick through recursion to generate the complimenting strand. The separation is done by keeping everything in a 2D array to make things more efficient and than printing everything out through a function, to maximize efficiency of the code.
